vscode自动补全hexo特殊标签语法

发表于 2021-08-26 更新于 2021-09-01 分类于 hexo ， vscode自动补全 阅读次数：

随着hexo的深入学习，发现在hexo编写MarkDown文件的时候也有其特有的标签语法(Tag Plugins)，在撰写文章的时候加入这些标签能使排版更加直观，但随之而来的是提高了编写的难度，因此记录使用vscode如何做到hexo的自动补全，包括NexT主题的特殊标签的自动补全

工具

  • Visual Studio Code

版本

框架 版本
node 12.22.5
hexo 5.4.0
next 8.7.0
vscode 1.59.1

配置过程

配置用户片段

打开首选项 –> 用户片段 –>在弹出的搜索框输入markdown –> 点击打开 markdown.json

按照文档配置需要预设的联想，如何配置文档内容这里不赘述

可以参考《VS Code配置markdown代码片段》改文章配置

已配置好的关于hexo与hexi-next的特殊标签json

下载`markdown.json`

配置vscode settings.json

首选项 –> 设置 –> 找到settings.json选项

把[markdown]里的“editor.quickSuggestion”的false改为true

建议

建议下载Markdown All in One这款扩展插件，功能十分强大，有益于快速编写Markdown内容

该插件涵盖以下功能：

  • 提供了常用操作便利的快捷键
  • 支持目录快速补全
  • 一边书写一边预览
  • 可轻松转换为HTML文件和PDF文件
  • 优化了List editing的编辑
  • 可格式化table以及Task list
  • 支持特殊数学符号渲染

按照以上配置，就能完成自定义语法、自动补全的功能
